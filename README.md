## 1. Project Overview

This project focuses on solving a real-world problem faced by an
e-commerce platform, SmartCart. The platform currently applies the same
marketing strategy to all customers, without considering differences in
behavior.

The aim of this project is to analyze customer data and group customers
into meaningful segments using machine learning techniques.

------------------------------------------------------------------------

## 2. Problem Statement

SmartCart is a growing e-commerce platform that serves customers across
multiple regions. It has collected large amounts of customer data,
including demographics, purchasing behavior, and engagement patterns.

However, the platform currently uses generic marketing strategies for
all customers, without identifying different customer behavior patterns.
This results in inefficient marketing, missed opportunities to retain
valuable customers, and difficulty in identifying customers who are
likely to churn.

The objective of this project is to build an intelligent customer
segmentation system using unsupervised machine learning techniques.

------------------------------------------------------------------------

## 3. AI/ML Concepts Used

## 4. Dataset Description

The dataset contains customer-level data, where each row represents a
customer.

Key features include: - Demographic information (age, income, education,
marital status)\
- Spending on various product categories\
- Purchase frequency (web, store, catalog)\
- Website visits and engagement\
- Recency (time since last purchase)\
- Complaint history

## 5. Approach

1.  Data Cleaning and Preprocessing\

5.  Determining optimal clusters using Elbow Method\
